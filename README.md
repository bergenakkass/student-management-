# Student Management System (C)

##  Açıklama
Bu proje, C dili kullanılarak geliştirilmiş basit bir öğrenci yönetim sistemidir. 
Veriler binary dosya kullanılarak kalıcı şekilde saklanır.

##  Özellikler
- Öğrenci ekleme
- Listeleme
- ID ile arama
- Öğrenci silme
- Bilgi güncelleme
- Ortalama hesaplama

## Kullanılan Teknolojiler
- C programlama dili
- Binary file işlemleri (fopen, fread, fwrite)
- Struct veri yapısı

## Çalıştırma

```bash
gcc main.c -o student
./student
